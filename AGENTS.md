# BooklyFlow Project Instructions

You are helping build BooklyFlow, a smart appointment automation platform for small businesses.

Tech stack:
- Next.js App Router
- TypeScript
- Tailwind CSS
- Supabase later
- Clean reusable components

Project structure:
- This project uses the Next.js app directory.
- Keep pages under app/.
- Keep reusable components under components/.
- Keep mock data under lib/mock-data.ts if needed.
- Keep utility functions under lib/.

Coding rules:
- Use TypeScript everywhere.
- Use functional React components.
- Keep components small and reusable.
- Use clear names for variables, files, and functions.
- Prefer clean UI with responsive design.
- Use Tailwind CSS for styling.
- Do not add unnecessary libraries without asking.
- Do not change unrelated files.
- Explain important changes briefly.

Design direction:
- Modern calm business UI.
- Primary color: #7C3AED.
- Secondary color: #F472B6.
- Background: #F8F7FF.
- Text: #111827.
- Use rounded cards, soft shadows, spacious layout.
- Make the app look polished and professional.
